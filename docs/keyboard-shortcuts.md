# Keyboard Shortcuts

All shortcuts work globally across the app (not just when focused on a page).

## Page Navigation

| Key | Page |
|-----|------|
| **1** | Signal Chain |
| **2** | Presets |
| **3** | IR Browser |
| **4** | MIDI Configuration |
| **5** | Live Mode |
| **6** | AI Assistant |
| **7** | Settings |

These work from any page — no need to click back to navigation.

## Engine

| Shortcut | Action |
|----------|--------|
| **Space** | Toggle engine start / stop |

## Signal Chain

| Shortcut | Action |
|----------|--------|
| **Ctrl+Z** | Undo last signal chain change |
| **Ctrl+Shift+Z** | Redo last undone change |

Up to 50 undo steps are preserved.

## Settings Page

| Shortcut | Action |
|----------|--------|
| **Ctrl+S** | Save settings |

Only works when the Settings page is active. Settings must be dirty (changed)
for the save button to be enabled.
